Introduction

Introduction With an abundance of premium content, HBO Max holds great appeal for viewers. However, within the competitive streaming landscape, offering an exceptional user experience is crucial for subscriber retention and growth. This UX case study will uncover opportunities to optimize core elements of the HBO Max platform to better serve user needs.

Content Discovery

The search parameters in HBO Max are too expansive, as genres are organized in a more dynamic manner. For instance, within the umbrella of "Thriller," there exists a spectrum of sub-genres like suspense, action, horror, and supernatural.  Furthermore, the search functionality is constraining, as users are restricted to locating titles using precise title matches rather than being able to search based on a desired "feeling" or "mood."

After the user is registered, it takes a long time generate the rest of the following content, but here’s a catch. The user doesn’t know what the other content is because there is no information provided. Unlike Netflix, HBO Max doesn’t have “Lazy Loaded Titles” for the next content coming up. The user has no idea what is loading.

Takeaways

HBO Max has consistently been a pivotal platform for streaming exclusive content, contributing significantly to the entertainment landscape. While the HBO Max website maintains a commendable standard, I have identified a promising opportunity for improvement in the redesign of the mobile application.
